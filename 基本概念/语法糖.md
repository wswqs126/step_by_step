# 定义

**语法糖**(Syntactic Sugar)，又译为“糖衣语法”，指计算机语言中添加的某种语法，这种语法对语言的功能并没有影响，但是更方便程序员使用。   

> 简单来说：就是在不改变原有功能的情况下，用语义更友好的形式编写代码。“糖”可以理解为给点甜头。

# 举个例子

在下面两个循环中，`for`循环可以认为是`while`循环的语法糖，因为这两个循环功能相同，但`for`循环书写更简单、便于理解。   

```c
for (i = 0; i < 10; i++)
{
  //P
}
```

```c
i = 0;
while (i < 10)
{
  //P
  i++;
}
```
